# 🌌 Acerca del proyecto

**Event Attire Rental System** es una plataforma web enfocada en facilitar la renta de atuendos elegantes para ocasiones especiales de manera rápida, moderna y accesible.

El sistema permite:

- 👗 Explorar prendas elegantes
- 💍 Rentar vestimenta para bodas y eventos
- 🔍 Buscar atuendos rápidamente
- 💰 Filtrar por presupuesto
- 📅 Gestionar reservaciones
- 📋 Administrar historial de rentas

La plataforma fue diseñada para brindar una experiencia intuitiva tanto para clientes como para administradores del sistema.

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/your-repository/event-attire-rental-system.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd event-attire-rental-system
```

---

## 3️⃣ Configurar base de datos

Crear base de datos MySQL y configurar credenciales del backend.

---

## 4️⃣ Ejecutar backend

```bash
./mvnw spring-boot:run
```

o

```bash
gradle bootRun
```

---

## 5️⃣ Ejecutar frontend

```bash
npm install
npm run dev
```

---

## 6️⃣ Abrir aplicación

```bash
http://localhost:5173
```
