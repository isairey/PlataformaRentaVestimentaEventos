# Event Attire Rental System

An online platform for renting elegant attire for special occasions like engagements, weddings, and receptions. This system provides a convenient and affordable way to access high-quality event outfits.

---

## Features



- **Gender-Based Sorting**: Filter attire based on gender preferences (e.g., male, female).
- **Price Sorting**: Sort outfits by price (low to high) for easier selection within budget.
- **Search Functionality**: Quickly search for attire using keywords like cloth type.
- **Booking System**: Seamlessly book attire with confirmation of availability and rental details.
- **Rental Management**: View current and past rentals.

## System Architecture

The project follows a **frontend-backend architecture**:
1. **Frontend**: Built using React.js, providing a dynamic and user-friendly interface.
2. **Backend**: Developed with Spring Boot to handle API requests and database operations.
3. **Database**: MySQL stores user data, inventory, and rental orders.
